This is maven repository used by distributed cruisecontrol build for bluecove native libraries.

If required native libraries for released versions should extracted from distribution bluecove-$VERSION$.jar
The custom build native libraries for released versions can be taken from http://snapshot.bluecove.org/special-build/

The libraries in this directory produced by bluecove build process running on Windows XP/Mac OS X Leo/Linux Fedora 32-bit in VMware Server on pyx4j.com Linux build server.
